kleiner spickzettel git

neues repository

git init

initialisiert ein lokales git repository

git add .

gibt an was alles in den commit rein soll

. gibt an, dass alles in den commit soll

git commit -m “message”

hinterlegt einen message/ commit (man kann auch sagen: legt eine version an)

git push

schickt das lokale repository in das github repository

commits anzeigen

git log

zeigt alle bis dato erstellten commits ausführlich an

git log --oneline

zeigt alle bis dato erstellten commits in kurzform an

branches

git branch “name”

erstellt einen neuen branch

git branch -a

zeigt alle erstellten branches an

git branch -d “name”

löscht den ausgewählten branch

git checkout

wechseln zwischen den branches

anpassen

git pull

lädt die aktuelle version des repository von github herunter

git clone

kopiert ein fremdes projekt von github

git merge

ist dafür da um zwei branches zusammenzuführen
